API de Estados y Ciudades de Venezuela

Consulta los estados y ciudades de Venezuela.

Estados

    Ciudades

      Cómo usar la API

      Obtener todos los estados

      fetch('/api/states')
.then(response => response.json())
.then(states => console.log(states));

      Obtener todas las ciudades

      fetch('/api/cities')
.then(response => response.json())
.then(cities => console.log(cities));

      Filtrar ciudades por estado

      fetch('/api/cities?stateId=1')
.then(response => response.json())
.then(cities => console.log(cities));

      Desarrollado como API pública para consultar los estados y ciudades de Venezuela.
